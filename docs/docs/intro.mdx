## 1 主要内容

信息时代，网络已经成为空气一般的必需品——我们的生活离不开网络，却也因其普及性而忽视了网络运转背后复杂而精妙的实现，缺少获得理解、修复、改造网络经验的机会

我们希望借助《计算机网络》课程的探索过程，帮助你在交叉进行的实验与理论教学中自底向上理解计算机网络运转的机理，从对数据流转的最小单元的分析，逐步提升、扩展到大型网络的构建，将理论的记忆与学习转化为生活中可以实际应用的故障排除技巧和方法论，内化为自己实现与创新网络技术的灵感来源
