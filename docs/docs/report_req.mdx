## 实验报告提交

* **报告模板**：使用来自作业系统或本实验文档网站提供的，Markdown或Word版本均可，实验过程中请按照实验报告具体要求填写全部部分
* **心得体会**：避免空泛感想（不需要写小学生游记），可**简单**列出你在实验中遇到的问题和解决方法等，也可以写一些自己的分析、思考与总结，该部分填写即不会被扣分
* **文件格式**：提交实验报告前，请将实验报告导出为PDF格式，如使用Markdown版本，请注意导出后PDF的阅读友好性，避免出现全文居中、字体极小等问题
* **附件提交**：每个实验除了实验报告之外，可能还需要提供一些其它测试数据，请压缩后递交到作业系统中，采用自己的学号_名字_LabX (X是从1-8)命名，如果文件过大，请分成两个压缩包递交，分别采用学号_名字_LabX_1和学号_名字_LabX_2来命名，你可以将要提交的文件拖拽到这里检查命名是否符合要求（不会进行文件上传，请放心使用）

    <SubmissionChecker />

* **提交覆盖**：在截止时间前，你可以任意重新提交实验报告，但需要在提交后联系老师说明你需要保留全部提交中的哪一份，以确保以正确的实验报告作为评分基础
* **实验附图**：实验报告的附图禁止使用手机拍屏，必须使用合适的截图工具，如实验步骤要求你在图上进行标注，你必须在图中标注，或用文字单独描述要标注的内容，否则将损失分数

## 实验组队

* **Lab 1/4/5/6** 要求独立完成
* **Lab 2/3** 由于网络设备有限，可2-3人合作完成，按最终报告分别独立打分
* **Lab 7/8** 允许最多2人组队，团队共享除思考题部分外的分数；鼓励独立完成（对编程能力较强的同学而言可能效率更高），但独立完成没有奖励分数

组队完成实验时，请在实验报告中注明组队成员姓名；组队完成实验时，也需要独立完成思考题

## 实验诚信

为保证公平公正的学术环境，请同学们：

* **独立完成**: 实验项目及代码须独立完成，严禁直接照搬或借用他人的成果。严禁抄袭他人（包括往届学生在任何代码托管平台或社区发布的）代码
* **代码保密**: 禁止将个人实验代码以公开形式发布于任何代码托管平台（如 GitHub）或社区（如 CSDN、博客园等）
* **合理求助**: 在帮助同学时，请以启发和引导为主，不要直接提供完整的源代码
* **禁止代写**: 严禁使用他人或生成式人工智能工具提供的完整实验代码
* **独立思考**: 思考题的解答必须独立完成，严禁直接使用他人或生成式人工智能工具提供的完整答案
