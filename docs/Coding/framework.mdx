## 2 测试框架使用

### 2.1 关键词解释

* 测试服务地址： 测试服务运行的IP与端口号，测试页面需要通过该地址连接到测试服务，发起测试请求并根据响应展示结果

* Socket服务端地址：Socket服务端运行的IP与端口号，测试页面需要通过该地址连接到测试服务，发起测试请求并根据响应展示结果

* Web服务器地址：Web服务器运行的IP与端口号，测试页面需要通过该地址连接到测试服务，发起测试请求并根据响应展示结果
